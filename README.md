# Bounty Quests Redone - NG

* [Nexus Page](https://www.nexusmods.com/skyrimspecialedition/mods/109661)

### Requirements

* [XMake](https://xmake.io/) [2.7.8+]
* C++23 Compiler (MSVC)

## Build
```
xmake build
```
